# Shi - Japanese Products E-commerce Web Site

**Shi** — Yapon məhsullarının onlayn nümayişi və satışı üçün hazırlanmış müasir və estetik bir **E-commerce (Elektron Ticarət)** frontend veb saytıdır

Bu layihənin əsas məqsədi istifadəçilərə rahat, vizual olaraq cəlbedici və sürətli bir onlayn alış-veriş təcrübəsi təqdim etməkdir. Layihə tam olaraq müasir frontend standartlarına uyğun şəkildə yığılmışdır

---

## Özəlliklər (Features)

* **Məhsulların Nümayişi:** Yapon məhsullarının interaktiv və səliqəli şəkildə istifadəçilərə təqdim olunması
* **Kateqoriyalar üzrə Axtarış:** İstifadəçilərin istədikləri məhsul növlərini asanlıqla tapması üçün dinamik filterləmə və axtarış funksiyası.
* **Səbət Sistemi (Shopping Cart):** Bəyənilən məhsulların real vaxt rejimində səbətə əlavə edilməsi, silinməsi və ümumi qiymətin izlənilməsi.
* **Responziv Dizayn:** Həm kompüter, həm də mobil cihazlarda (media) tam ekran uyğunluğu.

---

## İstifadə Olunan Texnologiyalar (Tech Stack)

* **HTML5** — Veb saytın semantik strukturunun qurulması üçün].
* **CSS3** — Vizual dizayn, rəng palitrası, animasiyalar və responsivlik üçün
* **JavaScript (ES6+)** — Səbət sistemi, axtarış mexanizmi və digər dinamik interaktiv funksiyaları təmin etmək üçün
* **Figma** — Layihənin kodlaşdırılma mərhələsindən əvvəl UI/UX dizaynının hazırlanması və piksel-piksel koda köçürülməsi üçün

---

## Layihənin İşə Salınması (How to Run)

Bu layihə tamamilə frontend (müştəri tərəfli) texnologiyaları ilə yığıldığı üçün heç bir əlavə server və ya db quraşdırmağa ehtiyac yoxdur
